# ｍやるゼ！ ChatGPT 画像生成プロンプト集（実運用版）

設計書 v4.4 §49 のプロンプト集を、**ChatGPT にコピペすればそのまま生成できる形**に整形した実運用ガイド。

---

## 0. ChatGPT 側の事前準備

1. **ChatGPT Plus / Pro プラン**にログイン（GPT-4o / GPT Image 機能利用）
2. 新規チャットを開き、モデルが GPT-4o になっていることを確認
3. 1 枚生成するごとに**新規チャット**を立てると、前画像の影響を受けず一貫した品質で出る
4. 画像は右クリック → 「画像を保存」で PNG ダウンロード
5. ファイル名は **§49-6 の命名規則**に従って保存：`{用途}_{シーン}_{バリエーション}.png`

# §A. LP ヒーロー画像（1920×1080・実写風）

## §A-1. ヒーロー候補A：訪問診療シーン

**ChatGPT への指示文（このまま貼り付け）：**

```
以下の指示通り、医療系SaaSのランディングページヒーロー画像を1枚生成してください。

シーン：
- 明るく開放的な日本の高齢者の自宅リビング
- 窓から差し込む柔らかな自然光
- 中年のアジア人女性看護師（白いユニフォーム着用）が、笑顔で年配の女性患者（70代・アジア人）の手を握って会話している
- 看護師の手元にはタブレット端末（具体的なブランド名・ロゴは映さない）
- 背景：暖かい木目調のインテリア（控えめ）

トーン：
- 清潔感・信頼感・温かみを両立
- 医療系の硬さと家庭的な親しみのバランス

構図：
- 横長 16:9 アスペクト比（1920×1080想定）
- 画像左下に文字オーバーレイ用の余白を確保

スタイル：
- 写真のような実写風（photorealistic）
- シャープでプロフェッショナルな撮影品質
- 浅い被写界深度（背景は柔らかくぼかし）

カラーパレット：
- パステルブルー(#0369a1系)と白を基調
- わずかに暖色のアクセント

禁止事項：
- 医療機器の具体的ブランド名・ロゴを映さない
- 既存有名人や特定アジア人著名人の顔を模倣しない
- 過度に演出的な表情（過剰な笑顔・芝居がかった所作）を避ける
```

**チェック観点：**
- [ ] 看護師が「白いユニフォーム」になっているか（スクラブだと別シーン）
- [ ] 患者が「年配の女性」になっているか
- [ ] 左下に文字を載せる余白があるか
- [ ] 医療機器のロゴが映り込んでいないか
- [ ] 全体トーンが青系＋白＋暖色アクセントになっているか

**保存ファイル名：** `hero_visit_v1.png`（気に入らなければ v2, v3 とリテイク）

**リテイク指示パターン：**
- もう少し明るく → 「自然光をもう一段明るく、ハイライトを強めにしてください」
- 患者を若く → 「患者の年齢を60代前半に変更してください」
- 構図変更 → 「人物を画面右側に寄せて、左側1/3を余白にしてください」

# §E. 共通禁止事項（全画像に適用）

ChatGPT が忘れがちなので、生成前に必ず以下を確認：

| 禁止事項 | 理由 |
|---|---|
| 既存有名人・実在著名人の顔の模倣 | 肖像権 |
| 後藤基温先生本人を模した顔の生成 | 本人写真は別途実写取得（§48-4） |
| 医療機器・タブレット・PCのブランドロゴ表示 | 商標権 |
| 具体的な患者名・病名・医療データの可読表示 | 個人情報 |
| Disney/Marvel/Pixar等の著作キャラ | 著作権 |
| 過度な肌の露出（マッサージシーン） | プロフェッショナル感維持 |
| グロテスク・痛そうな医療シーン | サービストーン維持 |
| 既存有名アプリのアイコンデザイン模倣 | 商標権 |

# §F. 生成後のワークフロー

1. **ダウンロード**：気に入った画像を保存（命名規則順守）
2. **配置**：`medadapt/assets/img/generated/` 配下に保存
3. **コミット**：
   ```bash
   git add assets/img/generated/
   git commit -m "feat(assets): ChatGPT 生成画像追加（hero / cards / icons）"
   git push origin main
   ```
4. **本番反映**：GitHub Pages 経由で 1〜3 分で `https://myaruze.tamjump.com/assets/img/generated/{filename}.png` でアクセス可能
5. **コード組み込み**：`index.html` / `app.html` 内の該当 `<img src="">` を差し替え or 追記

# §G. リテイク指示の汎用パターン

| やりたいこと | ChatGPT への追加指示 |
|---|---|
| 明るさ調整 | 「自然光をもう一段{明るく/暗く}してください」 |
| 人物の年齢調整 | 「{看護師/患者}の年齢を{20代/30代/40代/60代/70代}に変更してください」 |
| 構図変更 | 「人物を画面{左/中央/右}に寄せて、{反対側}1/3を余白にしてください」 |
| 表情変更 | 「{看護師/患者}の表情をもう少し{穏やか/笑顔控えめ/真剣}にしてください」 |
| カラー変更 | 「全体トーンをもう少し{暖色寄り/寒色寄り/白基調}にしてください」 |
| 要素削除 | 「{タブレット/書類/植物}を削除してください」 |
| 要素追加 | 「画面{左下/右上}に{時計/植物/カレンダー}を追加してください」 |
| 質感変更 | 「全体を{もう少しシネマティック/もう少し雑誌の表紙風/もう少しドキュメンタリー風}にしてください」 |
